# Guessing Game
This repository contains files for an interactive game using Python that teaches 5 to 7 year olds how to add, subtract, multiply and divide positive integers. The game ensures that the values selected for subtraction return positive answers and the values selected for division do not return fractional answers.

### Main Components:
1) GuessingGame.py - This script runs the GuessingGame
2) Submission.txt - This file contains a sheet of problems that can be submitted and solved by the program

### Instructions:
1) Run "py GuessingGame.py"
2) Select Mode; Addition, Subtraction, Multiplication, Division, Random Questions
